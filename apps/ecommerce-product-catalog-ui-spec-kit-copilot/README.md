# E-commerce Product Catalog UI — Spec Kit + Copilot

This case explores using **Spec Kit** with **GitHub Copilot** to generate an e-commerce product catalog UI from specifications.

## Important

Spec Kit requires configuration at the **root of the repository**. Due to this constraint, the actual implementation was carried out in a **separate, dedicated repository**.
