# LLM PoC Research

Welcome to the documentation for the **LLM PoC Research** project.

This research investigates whether LLMs can reliably accelerate proof of concept development across five distinct domains.

## Case Studies

| Case | Domain | Stack | Result |
|------|--------|-------|--------|
| [E-Commerce Product Catalog](ecommerce-product-catalog-ui/case-report.md) | Frontend Web | React, TypeScript, Vite | Scaffolding fast, UI failed |
| [Financial Tracker Mobile App](fintech-personal-finance-overview/case-report.md) | Mobile | React Native, Expo, TypeScript | Not viable (200+ iterations) |
| [Public API Integration](public-api-integration/case-report.md) | Backend API | ASP.NET, Azure APIM, Entra ID | Strong (<5 iterations) |
| [Private API Integration](private-api-integration/case-report.md) | Backend API | ASP.NET, private NuGet | Marginal (25+ iterations) |
| [AI-Powered Feature](ai-powered-feature/case-report.md) | Full Stack + AI | ASP.NET, React, Azure OpenAI | Mixed |
| [SpecKit Evaluation](ecommerce-product-catalog-ui-spec-kit-copilot/case-report.md) | Alternative Tool | SpecKit | Not suitable for PoC |
